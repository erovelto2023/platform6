# Affiliate Pages - Page Builder Guide

## Overview
The Page Builder is a drag-and-drop interface for creating high-converting affiliate review pages with AI-generated content.

## Features

### 1. **Drag-and-Drop Interface**
- Reorder sections by dragging the grip handle
- Visual feedback during dragging
- Auto-save on reorder

### 2. **Section Management**

#### **Add Sections**
- Click "Add Section" at the bottom
- Choose from 12 section types:
  - Hero Section
  - Introduction
  - Features Grid
  - Pros & Cons
  - Comparison Table
  - Pricing Table
  - Coupon/Deal Box
  - Video Embed
  - Call to Action
  - FAQ
  - Author Box
  - Conclusion/Verdict

#### **Remove Sections**
- Click the trash icon on any section
- Confirms before deletion

#### **Enable/Disable Sections**
- Click the eye icon to toggle visibility
- Disabled sections are grayed out
- Won't appear in final page

### 3. **AI Content Generation**

#### **Per Section**
1. Click the sparkle icon on a section
2. (Optional) Add custom instructions
3. Click "Generate Content"
4. Preview appears below

#### **Custom Instructions**
- Guide the AI's output
- Examples:
  - "Focus on the free trial"
  - "Write in a casual, friendly tone"
  - "Emphasize mobile features"

### 4. **Page Templates**

#### **Software Review**
- Hero → Intro → Features → Pros/Cons → Pricing → CTA → FAQ → Author → Conclusion

#### **Product Review**
- Hero → Intro → Features → Pros/Cons → Coupon → Video → CTA → FAQ → Conclusion

#### **Comparison**
- Hero → Intro → Comparison Table → Pros/Cons → Pricing → CTA → Conclusion

#### **Bonus Page**
- Hero → Features → Coupon → CTA → FAQ

### 5. **Page Builder Interface**

#### **Header Bar**
- Page name and slug
- Template type badge
- Status badge
- Preview button (opens public page)
- Export button (coming soon)
- Generate All button (coming soon)

#### **Main Area**
- Section list with drag handles
- Each section shows:
  - Template name
  - Category badge
  - Generated status
  - Enable/disable toggle
  - Customize button
  - Delete button

#### **Sidebar**
- **Brand Info**: Name, type, knowledge sources
- **Page Stats**: Total sections, generated count, enabled count
- **Quick Tips**: Usage instructions

### 6. **Section States**

#### **Draft**
- No content generated yet
- Shows template description

#### **Generated**
- Green "Generated" badge
- Preview available when expanded
- Can regenerate with new instructions

#### **Disabled**
- Grayed out appearance
- Won't appear in final page
- Can re-enable anytime

### 7. **Workflow**

1. **Create Page**
   - Select brand
   - Choose template
   - Page created with default sections

2. **Customize Sections**
   - Reorder by dragging
   - Add/remove sections
   - Disable unwanted sections

3. **Generate Content**
   - Click sparkle icon per section
   - Add custom instructions
   - Generate content
   - Preview output

4. **Publish**
   - Preview full page
   - Export code (coming soon)
   - Publish to live URL

## Keyboard Shortcuts
- **Drag**: Click and hold grip handle
- **Delete**: Click trash icon
- **Toggle**: Click eye icon
- **Expand**: Click sparkle icon

## Tips for Best Results

### **Custom Instructions**
- Be specific about tone and style
- Mention key features to highlight
- Specify target audience
- Include any unique selling points

### **Section Order**
- Start with Hero to grab attention
- Follow with Introduction for context
- Use Features to showcase benefits
- Add Pros/Cons for credibility
- Include Pricing for transparency
- End with strong CTA

### **Knowledge Base**
- Add comprehensive brand info before building
- Include official sales pages
- Add competitor reviews for context
- Include video demos if available
